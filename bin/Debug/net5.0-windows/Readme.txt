1. Установить OpenSSL. После установки найти папку «include» в установленной директории. И нужно записать путь к этой папке куда-нибудь.
2. Установить утилиту «make для Windows».
3. Установить компилятор GCC.
4. Перезапустить ОС.
5. Зайти в «переменные среды». Убедиться в том, что в переменной PATH прописан путь к компилятору GCC и путь к «make для Windows» (это нужно, чтобы из консоли можно было просто вводить: gcc.exe / make.exe — вместо ввода полного пути).
6. Добавить путь к папке ssl и crypto (OpenSSL) в переменную PATH.
